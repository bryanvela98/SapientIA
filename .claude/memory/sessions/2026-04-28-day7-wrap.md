## Done — memory housekeeping
- `decisions.md` — two new ADRs:
  - **ADR-032** — Voice command grammar is fixed-keyword, not freeform.
    Predictable + testable + cheap + safe rationale. Documents the
    prefix-match precedence rule (length-sorted; longer phrases win,
    so `stop reading` beats `stop`).
